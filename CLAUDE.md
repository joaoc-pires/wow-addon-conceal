# CLAUDE.md

## Collaboration

No implementation without explicit instruction. The only trigger for writing or editing code is the user explicitly saying to do so (e.g. "go ahead", "implement it", "make that change"). None of the following are permission to act:

- Asking a question ("why did you…", "what would…", "how could…")
- Pointing out a problem or mistake ("you used the wrong library")
- Asking for an explanation or investigation
- Confirming you understand something
- Describing or agreeing with a plan ("that means we need to…", "so the changes are…")
- Outlining what you are about to do before doing it
- Any response that is not a direct instruction to implement

Explaining a plan and then executing it in the same turn is not allowed. Present the plan, then stop and wait.

When in doubt, answer and wait.
